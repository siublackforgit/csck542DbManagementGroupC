#Database Creation
Please execute university_schema_v4.sql file in your local database,  the sql already included data


#Prerequisites

Python 3.11.0
mysql-connector-python==9.6.0

#Install dependencies globally

navigate to the root path of your project foler -> Vs code -> Terminal -> pip install mysql-connector-python==9.6.0

#Db Config

Vs Code -> Open app/db/config.py -> changed the user and password to your local database config

#Main App Execution

Navigate to the root path of your project foler -> Vs code -> Terminal -> python main.py

#Unit Test Execution

Navigate to the root path of your project foler ->  python -m unittest app/unit_tests/test_query_execution.py
